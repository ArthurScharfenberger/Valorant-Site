# Valorant Brasil

Projeto front-end desenvolvido com foco em apresentar notícias sobre o cenário competitivo brasileiro de **Valorant** em uma interface moderna, organizada e com visual inspirado no universo do jogo.

## Sobre o projeto

O **Valorant Brasil** é uma página web criada para reunir conteúdos em destaque do cenário competitivo nacional, trazendo uma estrutura visual limpa, estilizada e temática.

A proposta do projeto foi praticar conceitos de desenvolvimento web com **HTML**, **CSS** e **JavaScript**, construindo uma interface atrativa, responsiva e com identidade visual alinhada ao jogo.
